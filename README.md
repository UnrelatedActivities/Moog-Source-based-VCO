# Moog-Source-based-VCO
Moog Source based Voltage Controlled Oscillator.   
The zipped file MOOG_SOURCE_VCO_3046_01.zip is an archived Kicad project file, made with Kicad Version 7. 
I have just installed Kicad on an old PC and this is the first time I am using it, so some things may require adjustments. 
The PCB was designed thinking on homemade PCBs using 100x100mm single sided boards. 
I suppose it can be easily edited by those with some experience with Kicad for proper PCB manufacture. 
More information about the VCO circuit can be found at  
http://www.youtube.com/@unrelatedactivities
